## Projetos Acadêmicos

### 🛒 Sistema de Gerenciamento de Pedidos (OMS)
Sistema distribuído em nuvem para processamento de pedidos com arquitetura escalável, integrando serviços backend e cloud. Utiliza Spring Boot no backend, infraestrutura na AWS (EC2, RDS, Lambda, API Gateway), containerização com Docker e roteamento via Nginx.

`Spring Boot` `AWS` `Docker` `Nginx` `Node.js`

---

### 🎓 Portal de Estágios
Plataforma full stack para gestão de vagas e candidaturas com CRUD completo, autenticação de usuários e documentação de API via Swagger. Desenvolvida com Spring Boot e JPA/Hibernate no backend e HTML/CSS/JavaScript no frontend.

`Spring Boot` `JPA` `Hibernate` `HTML/CSS` `JavaScript` `Swagger`

---

### 📊 Sistema de Streaming – Analytics Dashboard
Sistema de análise de dados com modelagem relacional e otimização de consultas SQL. Inclui dashboard Python para visualização das métricas do banco de dados.

`Python` `SQL` `Data Modeling` `Query Optimization`

---

### 🌡️ Dashboard IoT – Monitoramento com WhatsApp
Sistema IoT com sensores DHT22, comunicação via protocolo MQTT, visualização em Grafana e alertas em tempo real integrados ao WhatsApp via Node-RED.

`Node-RED` `MQTT` `Grafana` `DHT22`

---

### 🔐 Sistema de Gerenciamento de Incidentes de Segurança
Sistema de análise e gestão de incidentes com foco em modelagem de processos, gestão de riscos e segurança da informação, documentado com UML.

`UML` `Gestão de Riscos` `Segurança da Informação`
